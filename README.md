# Python News Hub — Deno Deploy Relay

این پروژه یک ریلی روی Deno Deploy است.

## اطلاعات

- **Target Server:** `https://aischool.eu.cc:443`
- **Project Name:** `deno-aischool.eu.cc-443`

## مراحل Deploy

۱. این پوشه رو روی GitHub پوش کن (یک ریپازیتوری جدید بساز).
۲. برو به https://dash.deno.com و با گیت‌هاب لاگین کن.
۳. روی **New Project** بزن.
۴. ریپازیتوری مربوطه رو انتخاب کن.
۵. در قسمت **Entrypoint** بنویس: `main.ts`
۶. **Install Step** و **Build Step** را خالی بذار.
۷. روی **Deploy** بزن.

بعد از deploy، دامنه‌ای مثل `deno-aischool.eu.cc-443.deno.dev` بهت میده.

## استفاده در V2Ray

از کانفیگ‌ساز HTML استفاده کن و دامنه `deno-aischool.eu.cc-443.deno.dev` رو وارد کن.
